# 트래블프레임 개인정보처리방침

시행일: 2026년 5월 13일

트래블프레임은 사용자의 사진 촬영, 구도 편집, 이미지 저장 기능을 제공하는 앱입니다. 본 개인정보처리방침은 트래블프레임이 어떤 권한을 사용하고, 사용자의 정보를 어떻게 처리하는지 설명합니다.

## 1. 수집하는 개인정보

트래블프레임은 회원가입, 로그인, 이름, 이메일, 연락처 등 사용자를 직접 식별할 수 있는 개인정보를 수집하지 않습니다.

## 2. 앱 권한 사용 목적

트래블프레임은 앱 기능 제공을 위해 다음 권한을 사용할 수 있습니다.

- 카메라 권한: 사용자가 앱 안에서 사진을 촬영하고 구도 가이드를 확인할 수 있도록 사용합니다.
- 사진 및 미디어 접근 권한: 사용자가 기준 사진을 불러오거나, 촬영 사진과 편집 결과를 저장할 수 있도록 사용합니다.
- 파일 접근 권한: 사용자가 선택한 폴더에 편집 이미지를 저장할 수 있도록 사용합니다.

위 권한은 사용자가 직접 기능을 실행할 때만 요청되며, 허용하지 않아도 앱의 일부 기능은 계속 사용할 수 있습니다.

## 3. 사진 및 파일 처리

사용자가 촬영하거나 선택한 사진, 편집한 이미지, 이미지 묶음 작업물은 사용자의 기기 내부 또는 사용자가 선택한 저장 폴더에 저장됩니다.

트래블프레임은 사용자의 사진, 영상, 파일을 외부 서버로 자동 전송하지 않습니다. 현재 앱의 이미지 저장 및 편집 기능은 기기 안에서 처리됩니다.

단, 향후 사용자가 명시적으로 영상 렌더링 기능을 사용할 경우, 선택한 사진과 설정값이 영상 생성을 위해 별도 렌더 서버로 전송될 수 있습니다. 해당 기능이 제공되는 경우 앱 화면에서 별도로 안내합니다.

## 4. 제3자 제공

트래블프레임은 사용자의 개인정보, 사진, 영상, 파일을 제3자에게 판매하거나 제공하지 않습니다.

## 5. 보관 및 삭제

앱 안에 저장된 사진과 작업물은 사용자의 기기에 보관됩니다. 사용자는 앱 안의 삭제 기능을 통해 저장된 사진과 작업물을 삭제할 수 있습니다. 앱을 삭제하면 앱 내부 저장소에 저장된 데이터도 함께 삭제될 수 있습니다.

사용자가 핸드폰 앨범 또는 다운로드 폴더에 저장한 파일은 앱 삭제 후에도 기기에 남을 수 있으며, 사용자가 직접 삭제할 수 있습니다.

## 6. 아동 개인정보

트래블프레임은 아동으로부터 개인정보를 의도적으로 수집하지 않습니다. 보호자가 아동의 개인정보 관련 문의가 필요한 경우 아래 연락처로 문의할 수 있습니다.
