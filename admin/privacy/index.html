개인정보처리방침

시행일: 2026년 5월 15일

트래블프레임은 사진 촬영, 구도 편집, 영상 만들기, 이미지 및 영상 저장 기능을 제공하는 앱입니다. 본 개인정보처리방침은 앱이 어떤 권한과 데이터를 사용하고, 이를 어떻게 보관하고 삭제하는지 설명합니다.

1. 수집하거나 처리하는 정보

  • Firebase 로그인 정보: 이메일, 사용자 UID, 표시 이름, 로그인 제공자, 이메일 인증 여부
  • 앱 사용 데이터: 구독 상태, 결제 이벤트 기록, 백업 설정, 앱 설정값
  • 사용자가 직접 선택하거나 생성한 파일: 촬영 사진, 편집 이미지, 영상, 선택한 음악 파일
  • 기기 권한 정보: 카메라, 사진 및 미디어 접근, 파일 선택 권한

2. 정보 이용 목적

  • 카메라 촬영과 구도 가이드 표시
  • 사진 편집, 여러 사진을 이용한 영상 만들기, 이미지 및 영상 저장
  • 로그인 사용자 식별, 이메일 인증, 비밀번호 재설정
  • 구독 상태 확인, 무료/유료 기능 구분, 백업 기능 제공
  • 사용자가 요청한 클라우드 백업과 복원 기능 제공

3. 사진, 영상, 음악 파일 처리

기본적으로 촬영 사진과 편집 결과는 사용자 기기 또는 앱 내부 저장소에 저장됩니다. 사용자가 직접 백업 기능을 켜거나 파일 저장 기능을 실행한 경우에만 선택한 파일이 Firebase Storage 또는 기기 앨범에 저장될 수 있습니다.

사용자가 선택한 음악 파일은 영상 미리보기와 영상 만들기 기능을 위해 앱에서 처리됩니다. 사용자가 직접 선택한 파일 외의 음악 파일을 임의로 수집하지 않습니다.

4. 제3자 제공 및 외부 서비스

트래블프레임은 사용자의 개인정보, 사진, 영상, 음악 파일을 판매하지 않습니다. 앱 운영을 위해 Firebase Authentication, Firestore, Storage 등 Google Firebase 서비스를 사용할 수 있으며, 해당 서비스는 앱 기능 제공을 위한 처리자로 사용됩니다.

5. 보관 및 삭제

앱 내부 데이터는 사용자가 앱에서 삭제하거나 계정 삭제를 요청할 때 삭제할 수 있습니다. 클라우드 백업을 사용하는 경우 구독 기간이 만료되면 추가 백업이 제한될 수 있으며, 만료 후 3개월이 지나면 백업 데이터가 삭제될 수 있습니다.

계정 및 데이터 삭제 요청 안내

6. 아동 개인정보

트래블프레임은 아동으로부터 개인정보를 의도적으로 수집하지 않습니다. 보호자가 아동의 개인정보 관련 문의가 필요한 경우 아래 연락처로 문의할 수 있습니다.

7. 개인정보처리방침 변경

본 개인정보처리방침은 앱 기능, 법령 또는 정책 변경에 따라 수정될 수 있습니다. 변경 사항은 본 페이지를 통해 고지합니다.
