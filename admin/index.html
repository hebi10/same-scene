계정 접근

관리자는 운영 페이지에 접근하고, 일반 사용자는 앱 이용을 위한 계정을 만들 수 있습니다. 관리자 권한은 회원가입만으로 부여되지 않습니다.

첫 관리자 등록

Firebase Console에서 직접 admins/관리자UID 문서를 만들고 email, role 값을 넣어 주세요.

관리자 로그인

Firebase Auth 계정으로 로그인합니다. Firestore의 admins/{uid} 문서가 있는 계정만 관리자 페이지에 접근할 수 있습니다.

일반 회원가입

앱을 처음 사용하는 고객용 계정을 만듭니다. 가입 후 이메일 인증 메일이 발송되며, 관리자 페이지 접근 권한은 부여되지 않습니다.

인증 메일이 오지 않을 때

이미 가입한 이메일이라면 위에 이메일과 비밀번호를 입력하고 인증 메일을 다시 보낼 수 있습니다.
